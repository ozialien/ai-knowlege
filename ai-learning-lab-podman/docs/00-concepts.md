# Concept map

## Inference
Run a local model and get tokens back.

## Embeddings
Turn text into vectors for similarity search.

## Vector database
Store vectors and search nearby vectors.

## RAG
Retrieve context first, then generate an answer grounded in that context.

## Agent graph
A workflow of nodes like planner -> tool -> responder.

## Memory and checkpoints
Save thread state between steps so a workflow can resume later.

## Human-in-the-loop
Pause before risky actions or require approval.

## MCP shape
A clean boundary where tools and resources are exposed in a standard-friendly way.

## Multi-agent split
Separate retrieval, code analysis, and operations into explicit roles.

## Eval and observability
Log runs, compare changes, and catch regressions.
